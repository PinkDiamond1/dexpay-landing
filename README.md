## Stack

- [GatsbyJs](https://www.gatsbyjs.org/)

## To use

- Fork or download this repository
- Ready!

To change site config `./utils/config.js`

## Setup

Run:

```
yarn install
```

## Development

To start development server

```
yarn start
```

## Deployment

```
yarn run build
yarn serve
```
